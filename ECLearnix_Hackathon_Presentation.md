## Slide 3: Data-Driven Discovery Process
**Our Analytical Methodology**

🔍 **Phase 1: Exploratory Data Analysis**
- 10,000 user records across 16 key dimensions
- User behavior patterns by demographics, platform source, engagement
- Event participation analysis by region and type

🤖 **Phase 2: Machine Learning Clustering**
- K-Means algorithm with standardized features
- Elbow method optimization for 3 distinct user segments
- Feature importance analysis for behavioral insights

📈 **Phase 3: Predictive Modeling**
- Random Forest for churn prediction (baseline model)
- Performance: 66.6% accuracy - identified improvement opportunities

## Slide 4: User Segmentation Breakthrough
**Three Distinct User Personas Discovered**

### 🔴 **Cluster 0: "At-Risk Learners" (59.5% of users)**
- **5,951 users | 32.18% churn rate**
- Low app engagement, minimal time spent
- **Action Needed**: Immediate re-engagement campaigns

### 🟢 **Cluster 1: "Loyal Champions" (20% of users)**  
- **1,997 users | 30.70% churn rate (lowest)**
- High engagement, newsletter subscribers
- **Opportunity**: Upselling and referral programs

### 🟡 **Cluster 2: "Course Completers" (20.5% of users)**
- **2,052 users | 32.55% churn rate**
- Complete courses but still leave - engagement gap
- **Strategy**: Post-completion retention programs
